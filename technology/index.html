Iceberg Catalog Sync

↗ GitHub

Enterprise-grade Apache Iceberg catalog replication with full metadata chain rewrite, OAuth 2.0 authentication, and fine-grained OPA policy enforcement. Solves the core problem: Iceberg metadata embeds absolute storage URIs — copying Parquet files without rewriting metadata leaves every on-prem table pointing back at the cloud source.

  • Binary-level Avro metadata rewriting for cross-cloud failover (AWS → Azure)
  • Pluggable storage backends: S3, ADLS Gen2, GCS, MinIO
  • OAuth 2.0 (RS256 JWT) + Open Policy Agent for fine-grained access control
  • Airflow operators for pipeline orchestration
  • Nessie catalog with PostgreSQL version store
Python Iceberg Airflow OPA OAuth 2.0 Nessie

Fabric Workspace Provisioner Python · FastAPI 
from fastapi import FastAPI, Depends
from app.auth import validate_token
from app.fabric_client import FabricClient

app = FastAPI()

@app.post("/workspaces")
async def provision_workspace(
    spec: WorkspaceSpec,
    token: str = Depends(validate_token)
):
    client = FabricClient()
    # Idempotent: find or create by display name
    workspace = client.get_or_create(
        display_name=spec.display_name,
        capacity_id=spec.capacity_id,
        domain_id=spec.domain_id,
    )
    # Reconcile AD group role assignments
    client.reconcile_roles(
        workspace_id=workspace.id,
        desired_roles=spec.role_assignments,
    )
    return {"workspace_id": workspace.id, "status": "reconciled"}
Iceberg Cross-Cloud Sync Python · Iceberg 
from catalog_sync.rewriter import MetadataRewriter
from catalog_sync.storage import StorageBackend

def sync_table(source: str, target: str, table: str):
    # Read source metadata chain (manifest list → manifests)
    rewriter = MetadataRewriter(
        source_backend=StorageBackend.from_uri(source),
        target_backend=StorageBackend.from_uri(target),
    )
    # Binary Avro rewrite — every URI repointed to target
    new_metadata_location = rewriter.rewrite_and_copy(
        table_name=table,
        uri_mapping={source: target},
    )
    # Register in target catalog with JWT auth
    catalog.register_table(
        table, new_metadata_location, bearer_token=jwt
    )

Multi-Cloud Iceberg Medallion PySpark · Iceberg 
from pyspark.sql import SparkSession
from config import get_catalog_config

def run_bronze_to_silver(spark: SparkSession, env: str):
    cfg = get_catalog_config(env)  # AWS / Azure / Local

    bronze = spark.read.format("iceberg") \
        .load(f"{cfg.catalog}.bronze.orders")

    silver = bronze \
        .filter("o_orderstatus != 'F'") \
        .withColumn("ingest_ts", current_timestamp()) \
        .dropDuplicates(["o_orderkey"])

    silver.writeTo(f"{cfg.catalog}.silver.orders") \
        .using("iceberg") \
        .tableProperty("format-version", "2") \
        .createOrReplace()
